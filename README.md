# Acidliner

Acidliner is a browser-based MIDI generator for 303-style basslines (TB-303 / TD-3 friendly).

https://nordl3.github.io/AcidLiner/AcidLiner.html

It generates short 1–2 bar loops with accents, slides (via legato overlap), ties (held notes), plus optional 8-bar evolving phrases. Export the MIDI and drop it straight into Ableton Live (or any DAW).

## Features

- Style DNA presets + Rhythm DNA presets
- Key + scale/mode selection
- 1–2 bar loop export (`.mid`)
- Optional 8-bar phrase export (`.mid`)
- Slides via configurable note overlap (ms)
- Accents via velocity
- Ties as held notes (no retrigger)
- Candidate generation with basic quality scoring
- Simple in-browser preview audio
